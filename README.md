## üöÄ How to Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/0monish/Recreate-Thirty-Six-Studio.git
   cd Recreate-Thirty-Six-Studio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
